# おねがいアイプリ カードビューアー

「おねがいアイプリ」のカード画像を、スマホやPCのブラウザで見やすく閲覧するためのシンプルなカードビューアーです。

カードを1枚ずつ大きく表示したり、一覧表示に切り替えたりできます。カード表面・裏面の切り替えにも対応しています。

## 特徴

- ブラウザだけで動作します
- スマホ表示を意識した画面レイアウトです
- カードを1枚ずつ大きく表示できます
- 前後ボタンでカードを切り替えられます
- カードをタップして表面・裏面を切り替えられます
- 一覧表示でカードをまとめて確認できます
- タブでカードの種類を切り替えられます
- レアリティごとに見た目の演出があります

## 画面でできること

### カードを1枚ずつ見る

画面中央にカードが大きく表示されます。

左右のボタンで、前のカード・次のカードへ移動できます。

### カードの表面・裏面を切り替える

カードをタップすると、表面と裏面が切り替わります。

### カード一覧を見る

画面上部の「一覧」ボタンを押すと、カードをグリッド形式で一覧表示できます。

一覧からカードを選ぶと、そのカードを大きく表示できます。

### カードの種類を切り替える

上部のタブから、表示するカードの種類を切り替えられます。

現在用意しているタブは以下です。

- 全て
- 1だん
- 2だん
- SPカード
- グミ
- MFカード

```text
.

```

`index.html` の中に、画面デザイン、操作処理、カードデータがまとめて書かれています。

## 使い方

### 1. ローカルで開く

このリポジトリをダウンロードまたはクローンします。

```bash
git clone https://github.com/IzminGit/aipri-card-view.git
```

フォルダ内の `index.html` をブラウザで開くと利用できます。

特別なビルド作業やインストールは不要です。

### 2. GitHub Pagesで公開する

GitHub Pagesを使うと、Webサイトとして公開できます。

1. GitHubでこのリポジトリを開く
2. `Settings` を開く
3. `Pages` を開く
4. `Build and deployment` の `Source` で `Deploy from a branch` を選ぶ
5. Branch に `main`、フォルダに `/root` を選ぶ
6. `Save` を押す

しばらくすると、GitHub PagesのURLが表示されます。

## 技術メモ

このアプリは、フレームワークを使わない静的Webアプリです。

使用している主な技術は以下です。

- HTML
- CSS
- JavaScript

React、Vue、Vite、Node.jsなどは使用していません。そのため、ファイルをブラウザで開くだけで動作します。

## 開発・修正の方針

小さな修正であれば、現在の `index.html` を直接編集する形で問題ありません。

機能が増えてきた場合は、以下のようにファイルを分割すると管理しやすくなります。

```text
.
├── index.html
├── style.css
├── script.js
└── cards.js
```

## 注意事項

カード画像は外部サイト上の画像URLを参照しています。

画像URLの変更や配信停止があると、カード画像が表示されなくなる可能性があります。

また、このリポジトリは個人利用・学習目的のカードビューアーとして作成しています。
