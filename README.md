# おねがいアイプリ カードビューアー

「おねがいアイプリ」のカード画像を、スマホやPCのブラウザで見やすく閲覧するためのシンプルなカードビューアーです。

カードを1枚ずつ大きく表示したり、一覧表示に切り替えたりできます。カード表面・裏面の切り替えにも対応しています。

## スクリーンショット

アプリの画面イメージです。

> 画像ファイルを `docs/screenshot-card-view.png` として追加すると、以下に表示されます。

![カードビューアーのスクリーンショット](docs/screenshot-card-view.png)

スクリーンショットを追加する手順は以下です。

1. アプリ画面をブラウザで開く
2. スクリーンショットを撮影する
3. 画像名を `screenshot-card-view.png` にする
4. リポジトリ内に `docs` フォルダを作成する
5. `docs/screenshot-card-view.png` として画像を配置する
6. GitHubにコミットする

## 特徴

- ブラウザだけで動作します
- スマホ表示を意識した画面レイアウトです
- カードを1枚ずつ大きく表示できます
- 前後ボタンでカードを切り替えられます
- カードをタップして表面・裏面を切り替えられます
- 一覧表示でカードをまとめて確認できます
- タブでカードの種類を切り替えられます
- レアリティごとに見た目の演出があります

## 画面でできること

### カードを1枚ずつ見る

画面中央にカードが大きく表示されます。

左右のボタンで、前のカード・次のカードへ移動できます。

### カードの表面・裏面を切り替える

カードをタップすると、表面と裏面が切り替わります。

### カード一覧を見る

画面上部の「一覧」ボタンを押すと、カードをグリッド形式で一覧表示できます。

一覧からカードを選ぶと、そのカードを大きく表示できます。

### カードの種類を切り替える

上部のタブから、表示するカードの種類を切り替えられます。

現在用意しているタブは以下です。

- 全て
- 1だん
- 2だん
- 3だん
- SPカード
- グミ
- MFカード

## ファイル構成

現在は、ほぼ1ファイルで構成されています。

```text
.
├── index.html   # アプリ本体。HTML / CSS / JavaScript / カードデータを含む
├── docs/
│   └── screenshot-card-view.png   # README用スクリーンショット
└── README.md    # この説明ファイル
```

`index.html` の中に、画面デザイン、操作処理、カードデータがまとめて書かれています。

## 使い方

### 1. ローカルで開く

このリポジトリをダウンロードまたはクローンします。

```bash
git clone https://github.com/IzminGit/aipri-card-view.git
```

ローカルサーバーを起動します。

```bash
cd aipri-card-view
python3 -m http.server
```

ブラウザで [http://localhost:8000](http://localhost:8000) を開きます。カードデータは `cards.json` から読み込むため、`index.html` を直接開くのではなくローカルサーバー経由で確認してください。

### 2. カードデータを更新・照合する

カード定義は `cards.json` にまとめています。新しいカードを追加したら、公式カードリストとの差分を次のコマンドで確認できます。

```bash
node scripts/check-cards.mjs
```

未登録カード、公式に見つからないカード、名称・キャラクターの差分があればセットごとに表示し、終了コード1で知らせます。

### 2. GitHub Pagesで公開する

GitHub Pagesを使うと、Webサイトとして公開できます。

1. GitHubでこのリポジトリを開く
2. `Settings` を開く
3. `Pages` を開く
4. `Build and deployment` の `Source` で `Deploy from a branch` を選ぶ
5. Branch に `main`、フォルダに `/root` を選ぶ
6. `Save` を押す

しばらくすると、GitHub PagesのURLが表示されます。

## 技術メモ

このアプリは、フレームワークを使わない静的Webアプリです。

使用している主な技術は以下です。

- HTML
- CSS
- JavaScript

React、Vue、Vite、Node.jsなどは使用していません。そのため、ファイルをブラウザで開くだけで動作します。

## 開発・修正の方針

小さな修正であれば、現在の `index.html` を直接編集する形で問題ありません。

機能が増えてきた場合は、以下のようにファイルを分割すると管理しやすくなります。

```text
.
├── index.html
├── style.css
├── script.js
└── cards.js
```

## 注意事項

カード画像は外部サイト上の画像URLを参照しています。

画像URLの変更や配信停止があると、カード画像が表示されなくなる可能性があります。

また、このリポジトリは個人利用・学習目的のカードビューアーとして作成しています。
